# Startup Priority Tracker

Responsive to-do list application for desktop and mobile.

## Features
- Add tasks
- Prioritize tasks
- Categorize work
- Mark tasks complete
- Auto-save with localStorage
- Mobile responsive

## Run Locally

1. Install Node.js
2. Open terminal in project folder
3. Run:

npm install
npm run dev

4. Open:
http://localhost:5173

## Mobile Access

While running locally:
- Find your computer IP address
- On your phone connected to the same Wi‑Fi:
  http://YOUR-IP:5173
